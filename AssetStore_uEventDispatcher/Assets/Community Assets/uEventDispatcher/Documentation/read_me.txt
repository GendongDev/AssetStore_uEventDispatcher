
//--------------------------------------
//  OVERVIEW
//--------------------------------------

The packages facilitates decoupled communication between 2 or more objects within C# via the observer pattern. There are no dependencies. Any object type is compatible (i.e. not only MonoBehavior). 




//--------------------------------------
//  DEPENDENCIES
//--------------------------------------

* None


//--------------------------------------
//  INSTRUCTIONS
//--------------------------------------

* Open Unity 4.3.4 or later
* Open Assets/uEventDispatcher/Examples/Scenes/uEventDispatcherDemo.unity
* Enjoy!
*
* For production, you may remove Assets/uEventDispatcher/Examples/
